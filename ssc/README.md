# Мінісайт з JavaScript (Практична №7–8)

## Опис

Це багатосторінковий мінісайт, створений на HTML, CSS та JavaScript.
До існуючого сайту було додано клієнтську логіку: взаємодію з DOM, обробку подій, роботу з формами та localStorage.

## Реалізований функціонал

- 🔹 Підсвічування активної сторінки в навігації
- 🔹 Мобільне меню (бургер-кнопка)
- 🔹 Перемикач світлої/темної теми (збереження в localStorage)
- 🔹 Кнопка "Вгору"
- 🔹 Динамічний рік у footer
- 🔹 Акордеон
- 🔹 Модальне вікно (перегляд зображень)
- 🔹 Форма з:
  - валідацією
  - повідомленнями про помилки
  - лічильником символів
  - збереженням чернетки (localStorage)
  - обробкою через FormData

## Як запустити

1. Відкрити папку проєкту
2. Запустити файл `index.html` у браузері

## Додатково

- Дані форми зберігаються в localStorage
- Тема сайту запам’ятовується після перезавантаження
