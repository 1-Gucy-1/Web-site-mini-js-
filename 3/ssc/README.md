# Практична робота 3

## Основи HTML. Структура документа, контент, форми та медіа

## Опис проєкту

У цій практичній роботі було створено базовий багатосторінковий HTML-сайт без використання CSS та JavaScript.

Метою було опанування основ HTML:

- структура документа
- семантичні теги
- текстовий контент
- форми
- медіа (відео та аудіо)

Сайт складається з трьох сторінок:

- **Головна (index.html)**
- **Про мене (about.html)**
- **Контакти (contact.html)**

## Реалізовані можливості

### Структура HTML5

- Використано коректний каркас документа:
  - `<!DOCTYPE html>`
  - `<html lang="uk">`
  - `<head>` та `<body>`

- Додано:
  - `<meta charset="UTF-8">`
  - `<meta name="viewport">`
  - `<meta name="description">`

### Семантична розмітка

На всіх сторінках використано:

- `<header>`
- `<nav>`
- `<main>`
- `<footer>`

### Навігація

- Реалізовано меню з посиланнями між сторінками:
  - Головна
  - Про мене
  - Контакти

- Використано тег `<a>` з атрибутом `href`

### Контент (index.html)

- Додано:
  - заголовки (`<h1>`, `<h2>`)
  - абзаци (`<p>`)

- Використано:
  - `<strong>` — для виділення важливого тексту
  - `<em>` — для акценту

- Реалізовано списки:
  - маркований (`<ul>`)
  - нумерований (`<ol>`)

### Зображення

- Використано `<img>` з атрибутами:
  - `src`
  - `alt`

- Зображення обгорнуто в:
  - `<figure>`
  - `<figcaption>`

### Таблиця

- Створено таблицю з використанням:
  - `<table>`
  - `<caption>`
  - `<thead>`
  - `<tbody>`
  - `<th>`
  - `<td>`

### Форма (contact.html)

Реалізовано HTML-форму з:

- `<form>`
- `<label>` + `<input>`

Поля:

- ім’я (text, required)
- email (email, required)
- телефон (tel)
- тема (select)
- повідомлення (textarea)
- checkbox (згода)

Додано:

- радіокнопки (radio)
- кнопка submit

Використано HTML5-валідацію:

- `required`
- `minlength`
- `maxlength`

### Медіа (about.html)

- Додано відео:
  - `<video controls>`
  - кілька `<source>`

- Додано аудіо:
  - `<audio controls>`
  - кілька `<source>`

- Реалізовано fallback-текст
