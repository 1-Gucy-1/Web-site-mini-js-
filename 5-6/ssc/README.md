Основи CSS. Оформлення багатосторінкового HTML-сайту (Практична робота 5-6)

## Опис проєкту

У цій практичній роботі було реалізовано оформлення багатосторінкового HTML-сайту за допомогою CSS.
Проєкт базується на попередній практичній роботі (HTML-структура) та доповнений стилями для створення єдиного дизайну на всіх сторінках.

Сайт складається з трьох сторінок:

Головна (index.html)
Про мене (about.html)
Контакти (contact.html)

## Реалізовані можливості

Підключення стилів
Використано зовнішні CSS-файли:
style.css — основні стилі
responsive.css — адаптивність
Підключення виконано на всіх сторінках із правильними відносними шляхами

## Дизайн-система

Використано CSS-змінні (:root):
основний колір
фоновий колір
колір тексту
кольори поверхонь і меж
Задано глобальні стилі:
box-sizing: border-box
базовий шрифт
відступи та кольори

## Layout і структура

Реалізовано клас .container для центрування контенту
Використано .section для створення блоків із:
фоном
відступами
заокругленням

## Навігація, header, footer

Меню реалізовано через Flexbox
Додані стани:
hover
active
Прибрано стандартні стилі посилань
Header і footer оформлені в єдиному стилі

## Головна сторінка (index.html)

Стилізовано:
заголовки
абзаци
список
зображення
Реалізовано таблицю:
рамки
відступи
структура заголовків

## Сторінка "Про мене" (about.html)

Створено контентний блок із текстом
Додано:
відео (iframe)
аудіо
Використано Flexbox для розміщення тексту та зображення
Реалізовано блок із кількома зображеннями

## Сторінка "Контакти" (contact.html)

Форма оформлена як окремий блок
Стилізовано:
input
textarea
button
Додано стани:
:focus
:hover
Всі поля мають однаковий стиль

## Адаптивність

Використано media query:
@media (max-width: 768px)
Реалізовано:
вертикальне меню
адаптацію колонок (Flexbox → column)
коректне відображення зображень і таблиць

## Використані технології

HTML5
CSS3 (Flexbox, media queries, CSS variables)
